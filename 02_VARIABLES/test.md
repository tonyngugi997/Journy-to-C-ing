# C Variables Practice Question

Write a C program that:

1. Declares a `char` array called `city` with a maximum size of 50 characters.
2. Declares an `int` variable called `population`.
3. Declares a `float` variable called `average_temperature`.
4. Assigns the value `"Nakuru"` to `city`.
5. Assigns the value `2148000` to `population`.
6. Assigns the value `12.5` to `average_temperature`.
7. Prints them all in one line using a single `printf` statement.
